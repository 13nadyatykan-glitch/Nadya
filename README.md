# Training Planner

**Автор:** [Надежда] [Портнова]

## Описание программы

Графическое приложение для ведения дневника тренировок. Позволяет добавлять записи о тренировках с указанием даты, типа и длительности. Данные сохраняются локально в формате JSON. Доступна фильтрация по типу тренировки и дате.

## Технологии

*   **Язык:** Python 3.x.
*   **Библиотеки:** Tkinter (стандартная), json.
*   **Управление версиями:** Git.

## Инструкция по запуску

1.  Установите Python версии 3.6 или выше.
2.  Скачайте или клонируйте этот репозиторий.
3.  Откройте терминал в папке проекта и запустите приложение:
    ```bash
    python training_planner.py
    ```
4.  Приложение откроется в новом окне.

## Примеры использования и тесты

*   **Тест 1: Добавление тренировки**
    Введите корректные данные (например: Дата: `2026-05-05`, Тип: `Кардио`, Длительность: `45`) и нажмите «Добавить тренировку». Запись должна появиться в таблице внизу.
*   **Тест 2: Валидация даты**
    Попробуйте ввести дату в неверном формате (например: `05/05/26` или `пятое мая`). Приложение должно показать окно с ошибкой.
*   **Тест 3: Валидация длительности**
    Введите длительность как текст (`сорок пять`) или отрицательное число (`-10`). Приложение должно показать ошибку.
*   **Тест 4: Фильтрация по типу**
    Введите в поле фильтра слово `Кардио` и нажмите «Применить фильтр». В таблице должны остаться только тренировки этого типа.
*   **Тест 5: Сохранение данных**
    Закройте приложение. Откройте файл `trainings.json`. Убедитесь, что добавленные вами тренировки сохранились. Запустите приложение снова — данные должны загрузиться автоматически.
